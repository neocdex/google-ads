Instructions to run the code in XAMPP
--------------------------------------
1. Copy project directory (google-ads) under the root directory of Apache web server.
2. Set your customer parameters and OAuth2 'credentials in google_ads_php.ini' file.
3. Start your Apache web server if not running.
4. Open your browser and navigate to 'http://localhost/google-ads/form.php' (if your project directory is google-ads).
5. Enter keywords separeted by comma in textarea and click in submit button to search.
